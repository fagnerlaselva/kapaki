  <action>
Add GA4 custom event tracking to every interactive button/link across all components. Use `useGtag()` composable (auto-imported by nuxt-gtag module). Pattern for each component:

In `<script setup>`, add:
```typescript
const { gtag } = useGtag()
```

Then add a tracking function and bind it with `@click` on the `<a>` element. The `<a>` keeps its `href` — the `@click` fires the event WITHOUT `preventDefault()` so navigation still works.

**EXACT changes per component:**

### 1. HeroSection.vue
Add to script setup:
```typescript
const { gtag } = useGtag()
function trackWhatsApp() {
  gtag('event', 'whatsapp_click', { event_label: 'hero_cta', link_url: whatsappUrl })
}
function trackVerServicos() {
  gtag('event', 'ver_servicos_click', { event_label: 'hero_anchor' })
}
```
- On WhatsApp `<a>` add: `@click="trackWhatsApp"`
- On "Ver servicos" `<a>` add: `@click="trackVerServicos"`

Note: `whatsappUrl` is a prop, so destructure it from props or reference `props.whatsappUrl`:
```typescript
const props = defineProps<{ whatsappUrl: string }>()
const { gtag } = useGtag()
function trackWhatsApp() {
  gtag('event', 'whatsapp_click', { event_label: 'hero_cta', link_url: props.whatsappUrl })
}
```

### 2. ServiceCard.vue

const { gtag } = useGtag()
function trackServiceCta() {
