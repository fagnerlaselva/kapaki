**4. `app/composables/useScrollThreshold.ts`** — SSR-safe scroll composable:
```typescript
export function useScrollThreshold(px: number = 300) {
  const isVisible = ref(false)

  onMounted(() => {
    const handler = () => {
      isVisible.value = window.scrollY > px
    }
    window.addEventListener('scroll', handler, { passive: true })
    onUnmounted(() => window.removeEventListener('scroll', handler))
  })

  return { isVisible }
}
```

CRITICAL: `window.addEventListener` MUST be inside `onMounted` — accessing `window` outside causes SSR crash. Use `{ passive: true }` for scroll performance. Pair with `onUnmounted` cleanup to prevent memory leaks.
